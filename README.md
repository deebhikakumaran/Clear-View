
# Ecochain

A community-powered platform for tracking invisible pollution.



## Tech Stack

**Client:** 
- Vite
- React
- Tailwind CSS

**Server:** 
- Firebase


## Installation

Clone the repository

```bash
  git clone my-project
```

Navigate to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
